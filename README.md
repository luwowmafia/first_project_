 ## Анализ лояльности пользователей Яндекс Афиши
### Цель исследования
Выявить ключевые факторы, влияющие на лояльность пользователей сервиса «Яндекс Афиша», и разработать практические рекомендации для отделов маркетинга и продукта по удержанию клиентов и увеличению частоты повторных покупок.
### Задачи исследования
  1. Предобработка и очистка данных: привести данные к единым стандартам (валюта, типы), обработать пропуски и отфильтровать аномальные значения (выбросы), искажающие типичное поведение пользователей.
  2. Профилирование пользователей: сформировать агрегированные профили клиентов, выделив метрики «первого касания» (тип мероприятия, устройство, регион) и показатели финансовой активности (LTV, средний чек).
  3. Анализ факторов удержания (Retention Rate): изучить влияние характеристик первого заказа на вероятность возврата пользователя.
  4. Исследование финансового поведения: проанализировать структуру выручки и состава заказов в разрезе различных групп лояльности.
  5. Временной анализ: оценить влияние календарных факторов и интервалов между покупками на жизненный цикл пользователя.
  6. Корреляционный анализ: с помощью коэффициента ($\phi_k$) выявить наиболее значимые математические связи между признаками профиля и итоговым количеством заказов.
  7. Формирование рекомендаций: на основе полученных инсайтов подготовить конкретные шаги по работе с различными сегментами аудитории.
### Инструменты
Pandas, Matplotlib, Seaborn, Phi-K
### Как запустить
- Установить библиотеки: pip install -r requirements.txt.
- Создать .env на базе примера.
- Запустить ноутбук.
